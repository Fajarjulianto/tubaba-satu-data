# Satu Data Tubaba

**Satu Data Tubaba** adalah portal data terpadu resmi Kabupaten Tulang Bawang Barat yang berfungsi sebagai pusat data terbuka bagi publik. Platform ini menyediakan akses ke ratusan dataset statistik sektoral untuk mendukung transparansi, penelitian, dan pembangunan berbasis data di wilayah Tubaba.
